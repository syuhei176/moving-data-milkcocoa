Milkcocoaのデータストア移行スクリプト

Milkcocoaのデータストア移行スクリプトです。このプログラムを実行する前に、移行元と移行先でセキュリティルールを以下のようにしてください。


* {
  permit: all;
  rule: true;
}

以下を入力して、「開始」を押してください。

旧MilkcocoaのApp ID
移行元のデータストアの名前
新MilkcocoaのApp ID
